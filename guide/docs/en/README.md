## Quick Start

```bash
# Install
go get github.com/fanqie/tank-websocket-go-server

# Use in your code
import "github.com/fanqie/tank-websocket-go-server/pkg"

// Create a new WebSocket manager
manager := pkg.NewManager()

// Start the server
manager.Start(":8080")
```
